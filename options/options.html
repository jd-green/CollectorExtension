Web Page Metadata Sender

v1.0.0

A simple Chrome extension that sends the current web page's URL and basic metadata to a pre-configured webhook.

Features

  • Capture and send the current URL and metadata to a configured webhook
  • Support for multiple independent instances with custom icons
  • Visual feedback through temporary badges
  • Retry mechanism for failed requests
  • Detailed activity logging for troubleshooting
